text.mew151.net

This subdomain is set up for text-only internet versions of my main WWW site which can be located at https://www.mew151.net.

To view the text-only services, you will need an appropriate client for the protocol. Lagrange is a good GUI client that supports both protocols! Otherwise, I recommend Lynx for Gopher browsing. For other Gemini clients, you can view a list of clients on this page.

Services

Here are the text-only services that run on this subdomain:

  • Gopher: gopher://text.mew151.net
  • Gemini: gemini://text.mew151.net

Hymn of the Moles

Game Info
Name: Hymn of the Moles
Year: 2014
Made in: C++
Download Game (Windows)

Hymn of the Moles is a rhythm game based on The Residents' 1981 album, Mark of the Mole, and the unfinished Atari game that was supposed to be based on it. You play as a Mole, hired to work in the mines. While working, you sing songs and this creates the main gameplay: hit the arrow keys in time with the various notes on screen and the beat.

You can download the game here. This is a Windows EXE, but I have tested it with Wine on my Linux computer and it works fine!

Technologies

  • Programmed in: C++
  • OpenGL
  • SDL2
  • FMOD Ex

Roles and Development

This game was a solo project, made during the 29th Ludum Dare game jam. The theme for this jam was Beneath the Surface.

The game was written in C++ with OpenGL, SDL2, and FMOD. A lot of the library code was written ahead of time. An interesting fact about the engine code I'm using here: it was written before I had taken Linear Algebra and so a lot of the code doing transformations like scaling/rotation are doing them manually, rather than using matrices to perform linear transformations. In hindsight, it would have been much easier to just use matrix multiplication... but I didn't know better at the time.

The game uses music and characters by The Residents.

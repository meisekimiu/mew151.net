Virtual Vacation Zone+

Game Info
Name: Virtual Vacation Zone
Year: 2021
Made in: Pixi.js + React
Play Game on itch.io

Virtual Vacation Zone+ is an interactive poetic memoir, board game, and visual novel. Submit your own vacation memory before you embark on your own cerebral getaway. Play a simple board game to determine what emotions you feel next on your vaation, and maybe even read other player's vacation memories.

Halfway through the game, you can play the subgame Telemetry & Jeff a short visual novel about Telemetry trying to find herself some lunch.

You can play the game on Itch.io right now! Feel free to submit your own memories!

Technologies

  • Programmed in: TypeScript
  • Pixi.js
  • React
  • Ink / Ink.js
  • API Built In: PHP and MySQL

Roles and Development

This game was a solo project. This game was made in an environment similar to a Game Jam. Specifically it was made for Nicole He's Vacation Stream from July of 2021. The game was made in about a week, although it has been updated several times after the stream to add in the new vacation memories and scenarios as the world opened back up for travel.

The initial "web site" the game starts on is built in React, which then transitions over to the rest of the game which is built in the same custom Pixi.js-based game engine I used in Extreme Tax Masters. This visual novel segment of the game is scripted in Ink and uses a newer version of the visual novel system also used in Extreme Tax Masters.

The graphics for the visual novel segment, as well as the music and writing for the rest of the game, were also done by me in addition to all of the engineering work.
